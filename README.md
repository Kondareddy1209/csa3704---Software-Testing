# Software Testing Course Repository


A comprehensive repository for the **CSA3704 Software Testing** course, containing Java implementations, test cases, and practical exercises related to software testing methodologies and best practices.

## 📚 About

This repository serves as a collection of coursework, assignments, and projects for the Software Testing course (CSA3704). It demonstrates various testing concepts, frameworks, and techniques through hands-on Java implementations.

## 🎯 Learning Objectives

- Understand fundamental software testing principles and methodologies
- Implement unit tests, integration tests, and system tests
- Apply test-driven development (TDD) practices
- Explore various testing frameworks and tools in Java
- Develop skills in writing maintainable and effective test code

## 🛠️ Technology Stack

## 🚀 Getting Started

### Prerequisites

- Java Development Kit (JDK) 8 or higher
- Git
- Gradle (optional, for build automation)

### Clone the Repository

```bash
git clone https://github.com/Kondareddy1209/software-testing-course.git
cd software-testing-course
```

### Building the Project

```bash
# Using Gradle
gradle build

# Or compile manually
javac src/main/java/**/*.java
```

### Running Tests

```bash
# Using Gradle
gradle test

# Or run specific test files manually
java -cp . <TestClassName>
```
